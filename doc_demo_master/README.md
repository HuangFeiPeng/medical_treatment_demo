## 医生端目录结构：

- -common	     		公共css/js目录
- -components  		组件目录
- -js_sdk/wa-permission  权限目录
- -newSDK  			环信IM目录
- -pages  				页面目录
- -plugins  			自动更新目录
- -static  				图片目录
- -store  				缓存目录
- -utils  				工具类目录
- App.vue				页面入口文件
- main.js				是uni-app的入口文件
- manifest.json			文件是应用的配置文件
- pages.json			页面文件的路径
- uni.scss				方便整体控制应用的风格
